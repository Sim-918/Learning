해킹-관리자, 운영자가 의도하지 않은 동작을 일으키도록 하거나 주어진 권한 이상으로 수정,열람,복제을 가능하게 하는 용어
크래킹-악의적인 목적을 가진 해킹
리버스 엔지니어링-소프트웨어를 보호하기 위해 구현한 특정 로직을 회피하거나 건너 뛰도록 바이너리를 수정하는 과정
시스템 해킹 절차-   터미널 준비->자신의 컴퓨터 안전성 확보->해킹 대상 테스트->해킹 대상 OS확인->침입경로탐색->패스워드 또는 인증 프로세스 크래킹->
                    루트권한 획득->백도어 만들기->침입 흔적 지우기
웹 해킹 유형(디렉토리 이동 공격-공개 도메인에 나타나지 않은 비인가 파일이나 폴더에 접근하여 중요 정보를 탈취하는 방법
             서비스 거부 공격(DOS)-대량의 데이터를 한 곳에 집중으로 전송해 기능을 방해하는 공격
             스니핑-네트워크 중간에서 패킷 정보를 도청하는 해킹
             피싱-다수에게 이메일을 발송하여 웹 사이트 접속을 한 후,개인정보를 탈취하는 행위
             파밍-DNS를 탈취하거나 사용자들에게 진짜 사이트로 오인할 수 있는 가짜 사이트로 유인해 개인정보를 탈취하는 행위
             웹 사이트 훼손-웹 서버에 침입해 웹 사이트내용을 공격자가 원하는 내용으로 바꾸는 행위
             )
네트워크 해킹(스캐닝-탐색하다.
             스니핑-네트워크 중간에서 패킷 정보를 도청하는 해킹
             스푸핑-네트워크에서 가짜를 진짜인 것으로 속여 원하는 정보를 탈취하는 행위
             스머핑-IP나 ICPM의 특성을 이용하여 한 곳에 대량의 데이터를 집중적으로 보내 네트워크 불능을 만드는 행위 
             )
분산 서비스 거부 공격(DDOS)-분산된 공격지점에서 대량의 데이터를 한 곳에 집중적으로 전송해 공격하는 행위
지능형 지속 위협(APT)-'항상 집요하고 지속적인 위협' 대상을 특정한 후, 공격자가 목적을 달성하기 위해 
                     다양한 해킹 기법을 동원하여 충분한 시간을 가지고 지속적을 서서히 공격하는 것
암호공격-(제로데이 공격-보안 취약점이 발견됬는대 공표도 되기 전에 취약점을 이용해 해킹하는 공격
         암호문 단독 공격-공격자가 암호문만 가지고 있는 상황에서 공격하는 행위(무차별 대입 공격, 빈도수 분석)
         알려진 평문 공격-공격자가 일정 부분의 평문과 이에 대응하는 암호문을 가진 행태에서 공격하는 행위,
                         대부분 고전암호(카이사르,아핀,전치암호 등)가 공격으로 해독가능
         선택 암호문 공격-공격자가 많은 암호문에 대해 평문을 가지고 공격하는 행위
         )
패스워드 크래킹-패스워드를 알아내는 적극적 행위
키로깅-사용자의 키보드 움직임을 탐지해 정보를 탈취하는 행위
버퍼 오버플로우-메모리에 할당된 버퍼의 크기보다 많은 데이터를 입력해 프로그램 오류를 발생시키는 것(stack->LIFO)
스푸핑(IP스푸핑-IP주소를 속여 네트워크접속
      ARP스푸핑-IP주소를 MAC주소로 변환시키는 취약점을 이용해 MAC주소를 속여 원래 목적지로 가야될 패킷을 공격자로 전송시켜 정보를 탈취하는 행위
      DNS스푸핑-사용자가 IP주소를 도메인 이름으로 변환시키고 통신을 시도하지만 공격자가 DNS서버를 장악해 DNS캐시정보를 조작하여 엉뚱한 사이트로 
                연결시키는 행위)
악성코드(컴퓨터 바이러스-숨어서 전파되는 악성코드. 스스로 복제해 컴퓨터를 감염시킴
        웜-스스로 복제해 컴퓨터 감염. 네트워크 대역폭을 잠식하고 손상
        트로이 목마-정상적인 기능을 하는 프로그램으로 위장하고 있다가 트로이 목마가 실행되면 악성루틴이 컴퓨터에 설치되어 공격하는 것
        스파이 웨어-ActiveX 취약점을 이용해 사용자의 동의 없이 컴퓨터에 설치되어 컴퓨터의 정보를 수집하고 전송하는 악성코드

백도어-보안을 제거한 비밀통로

ICMP FLooding(인터넷 제어 메세지 프로토콜)-네트워크 브로드캐스트주소를 통해 특정 네트워크에 연결된 모든 컴퓨터로 패킷을 보낼 수
                                          있도록 설정된 네트워크장치를 이용한 공격
                  Ping FLooding-대상 시스템에 막대한 양의 ICMP패킷을 보내는 방법. 대상 시스템은 요청에 응답하느라
                                다른 일을 하지 못하도록 하는 공격
                  SYN FLooding-SYN패킷을 반복적으로 전송하여 시스템을 마비시키는 공격

      HTML Script injection-
      XSS(크로스 사이트 스크립팅)-악의적인 스크립트를 포함시켜 실행되게 유도 하는 것
